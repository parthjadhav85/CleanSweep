# CleanSweep 

An intelligent desktop organizer built with Python. It automatically scans your folder and sorts files into categories like Images, Documents, and Software.

### Features
- Modern Dark UI
- Automatic File Sorting
- Secure & Offline

### How to Run
1. Install Python
2. Run `pip install -r requirements.txt`
3. Run `python main.py`

Note: Windows Defender may flag this executable because it is not digitally signed. This is a common false positive for independent Python applications. You can view the full source code above to verify its safety.
